To follow along with labs, make a virtualized hacking environment. 
These are easy to maintain but can take a bit of time to set-up. 
Be sure virtualization is enabled in your BIOS.


1.) Download and Install Virtual Box

2.) Download and Install the Extension Pack

3.) Download and install the Parrot Security Virtual Image

through Virtual Box  ( Default Login is root:toor )
  
4.) Sign up with TryHackMe.com and set up OpenVPN




There are performance benefits to running Kali/Parrot on hardware via
a Dual Boot, if you would like to dual boot follow the following guide.
The instructions can be applied to either Kali or Parrot, as they are
both very similar under the hood.
